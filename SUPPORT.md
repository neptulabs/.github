# Support

For help with Neptu / neptulabs projects:

- **Usage questions** – Open a GitHub issue using the *Question* template.
- **Bugs** – Open a GitHub issue using the *Bug report* template and include steps to reproduce.
- **Feature requests** – Use the *Feature request* template.

For **security vulnerabilities**, **do not** open a public issue. Please follow the instructions in `SECURITY.md` and contact us at **security@neptu.com**.

For commercial support or enterprise engagements, contact: **hello@neptu.com**.
